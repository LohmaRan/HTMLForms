HTML Reference Sheet

Forms

Elements

Tag Name Description
<form> Form Section containing interactive controls for submitting information.
<label> Label Represents a caption for an item in a user interface.
<input> Input Creates interactive controls to accept data from the user.
<textarea> Text Area A multi-line plain text editing control.
<select> Select Provides a list of <option> elements that users must select a value from.
<datalist> Datalist Provides a list of <option> elements that users can either select or type their own value into.
<option> Option Defines an item contained in a <select> or <datalist> element.
